# Project 3 Report

This project explored keyword co-occurrence networks in scientific publications. The first step was to load the dataset and isolate the keyword columns while ignoring the article title. The keywords were converted to lowercase, and rows with no keyword data were removed. From there, a weighted adjacency matrix was created where each row and column represented a unique keyword, and each cell showed how many times two keywords appeared together in the same article. The matrix was made symmetric by incrementing both directions of each keyword pair.

Next, the adjacency matrix was converted into a weighted network using NetworkX. In this network, each node represented a keyword and each edge represented a co-occurrence relationship between two keywords. The edge weights reflected how frequently the keyword pairs appeared together in the dataset. After the network was built, two node metrics were computed: degree and strength. Degree measured how many unique keywords each node connected to, while strength measured the total sum of the edge weights connected to that node.

The results showed which keywords were the most central in the network. The top 10 nodes by degree highlighted the keywords that connected to the largest number of other topics, while the top 10 nodes by strength showed the keywords with the highest total co-occurrence frequency. The top 10 node pairs by weight identified the keyword pairs that appeared together most often, revealing the strongest thematic relationships in the dataset.

Finally, a scatter plot of degree versus strength was created to visualize the relationship between the two metrics. The graph showed that keywords with higher degrees often also had higher strengths, suggesting that the most broadly connected keywords were also the most frequently used across articles. Overall, the analysis demonstrated how network methods can be used to uncover patterns and central themes in scientific publication data.
